## **Technologies**
- Next.js
- Typescript
- TailwindCSS
- Mapbox
- Mapbox GL

## **Features**
- [x] Home page
- [x] Search page
- [x] Active Map
- [x] Responsive
- [x] Server Side Rendering
- [ ] Filters

## **How to use**
- Clone this repository
- Install dependencies
```bash
npm install
# or
yarn
```
- Run the development server
```bash
npm run dev
# or
yarn dev
```
